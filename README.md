# Aggregator

## Overview
**Aggregator** is a web-scraping framework used to extract 
the products from _AliExpress.com_, _Amazon.com_, and _Tap.az_.
Users are able to search any product. The web page will
scrape the link of the product and its price from mentioned
shopping sites, then display them on the page.

## Requirements
Python 3.6+
You can see the dependencies in requirements.txt file.

`pip install -r requirements.txt`

## Execute Aggregator
The project opens in Debug Mode. Run the command:

`python run.py`

Aggregator only supports Google Chrome.
